# 🎮 Tetris Game (Java)

![Java](https://img.shields.io/badge/Java-17%2B-orange?logo=java)
![Platform](https://img.shields.io/badge/Platform-Desktop-blue)
![Status](https://img.shields.io/badge/Status-Active-success)
![License](https://img.shields.io/badge/License-MIT-green)

A classic **Tetris game built using Java (Swing/AWT)**.  
This project demonstrates **game development logic, OOP design, and event-driven programming** in Java.

---

## ✨ Features

- 🎯 Classic Tetris gameplay mechanics  
- ⬇️ Smooth automatic piece dropping  
- ⚡ Hard drop (Space key)  
- 🔄 Piece rotation system  
- 🧠 Collision detection  
- 🧹 Line clearing system  
- 🏆 Score tracking system  
- 💾 High score save using file system  
- 🎮 Keyboard controls  

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| ⬅️ Left Arrow | Move left |
| ➡️ Right Arrow | Move right |
| ⬇️ Down Arrow | Soft drop |
| ⬆️ Up Arrow | Rotate |
| Space | Hard drop |
| P | Pause |

---

## 🛠️ Tech Stack

- Java (JDK 17+)
